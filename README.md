# MongoDB Command Executor (Flask)

A simple Flask web app that lets you run MongoDB commands (like `insertOne`, `find`, `updateOne`, and `deleteOne`) directly from your browser.

---

## Features
- Execute MongoDB CRUD commands in real-time  
- Simple web interface  
- Uses Flask + PyMongo  
- Connects to local MongoDB (`localhost:27017`)
